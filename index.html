The History of JavaScript

By: Minh Chau Ha

  • 1994:

    NetScape browser was launched

  • 1995:

    Brendan Eich invented JavaScript for NetScape

  • 1997:

    ECMAScript 1 (ES1) was launched

  • 1998:

    ECMAScript 2 (ES2) launched with editorial changes to ES1

  • 2009:

    ECMAScript 5 (ES5) sharpened up the standards with strict mode, JSON support, and more

  • 2009:

    Node.js launched to allow JavaScript to be executed outside the browser

  • 2010:

    AngularJS framework was released and grew due to its object oriented and reactive programming

  • 2013:

    React.JS library was released with functional programming model

  • 2014:

    Vue.js framework was released and grew due to its reactive programming model

  • 2015:

    ECMAScript 6 (ES6) was released with let, const, default paramater values, and more array methods

  • 2016:

    ECMAScript 7 (ES7) was released with the exponential operator and Array.prototype.includes

  • - Current

    ECMAScript 8 (ES8) and beyond have been release with ES2020 in staging
